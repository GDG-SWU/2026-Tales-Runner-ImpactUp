# 📚 Dual Tales (이중언어 동화책 만들기)

- SDGs 4 양질의 교육, SDGs 10 불평등 감소
  - 언어 격차 해소
  - 학습 접근성 향상
- AI 기반 다문화 가정을 위한 이중언어 동화 제작 서비스입니다.

## 🛠 기술 스택
- **Backend:** Java 17, Spring Boot 4.0.6, Spring Data JPA
- **Database:** MySQL
- **AI:** Gemini API, Google Cloud Translation API, Imagen
